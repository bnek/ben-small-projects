## Agent Hierarchy

### Task Queue System
- orchestrator.agent.md  — Infinite loop driver. Invokes supervisor repeatedly, never plans or acts.
- supervisor.agent.md    — Owns full task lifecycle: picks tasks, spawns worker sub-agents with complete context, reviews results, moves tasks to done.
- worker.agent.md        — Receives instructions via prompt from supervisor, executes the task, returns summary as response.
